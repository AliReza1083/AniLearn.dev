## Flexbox

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

## Gridbox I

```css
.container {
  display: grid;
  justify-content: center;
  align-items: center;
}
```

<ComponentPreview name="center-gridbox" />

## Gridbox II

```css
.container {
  display: grid;
  place-items: center;
}
```

<ComponentPreview name="center-gridbox-2" />

## Position

```css
div {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

<ComponentPreview name="center-position" />

## Margin

```css
div {
  width: 1500px;
  margin: auto;
}
```

<ComponentPreview name="center-margin" />

> When using Position or Margin for centering, we need to target the element not it's parent element.
