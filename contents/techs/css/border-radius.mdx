---
  title: "Border Radius"
  description: "If you want to add rounded effect to your shape, use border-radius."
  level: ""
  created_at: 2023-12-20
---

```css
div {
  border-radius: 0px 0px 0px 0px;
  /* top, right, bottom, left */
}
```

<ComponentPreview name="border-radius-index" />

You can use one value that effect the whole corners of an shape.

```css
div {
  border-radius: 0px;
}
```
