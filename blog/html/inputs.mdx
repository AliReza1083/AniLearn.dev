# HTML `<input>` Tag

## Definition and Usage

In HTML, an input element is used to create a form control that accepts user input. An input element can be used to create various types of form controls such as text boxes, checkboxes, radio buttons, email fields, password fields, and more.

The input element can have various attributes that define its behavior and appearance, such as **"type"** (which defines the type of input control), **"name"** (which gives the input a name to be submitted with the form data), **"placeholder"** (which displays placeholder text in the input field), "required" (which makes the input field required), and more.

Here is an example of using type, name, placeholder:

```html:index.html
<input type="text" placeholder="your name" name="username">
```

Here are the inputs which you can use them:

```html:index.html
<input type="button">
<input type="checkbox">
<input type="color">
<input type="date">
<input type="datetime-local">
<input type="email">
<input type="file">
<input type="hidden">
<input type="image">
<input type="month">
<input type="number">
<input type="password">
<input type="radio">
<input type="range">
<input type="reset">
<input type="search">
<input type="submit">
<input type="tel">
<input type="text"> <!-- default value -->
<input type="time">
<input type="url">
<input type="week">
```

## Part |

Let's learn about each of them.

### number

Users can input numeric values using the `<input>` element with the type="number" attribute.

```html:index.html
<input type="number" min="10" max="50">
<!-- The minimum value to accept for this input. -->
<!-- The maximum value to accept for this input. -->
```

### password

The `<input>` element with the 'password' type attribute is used to create a text box where the entered text is masked to hide the characters, commonly used for password input fields.

```html:index.html
<input type="password">
```

### submit

The `<input>` element with the submit type allows the user to submit the form by clicking on it.

```html:index.html
<input type="submit" value="submit">
```

### reset

The `<input>` which the type is reset will give ability for the user to reset the form.

```html:index.html
<input type="reset" value="reset">
```

### text

it creates a text input field.

```html:index.html
<input type="text">
```

### url

it creates a text input field where you must type a link.

```html:index.html
<input type="url">
```

### range

`<input>` elements of type range let the user specify a numeric value.

```html:index.html
<input type="range" min="0" max="10">
<!-- max = The greatest value in the range of permitted values. -->
<!-- min = The lowest value in the range of permitted values. -->
```

### tel

This element generates a text field where the user needs to input a phone number.

```html:index.html
<input type="tel">
```
