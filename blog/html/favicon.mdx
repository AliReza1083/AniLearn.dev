By the end of this blog, You will learn about Favicon.

If you see all the websites, they are having a logo beside their website's name, That is called _Favicon_.

**e.g**

- [Google](https://www.google.com)
- [AniLearn](https://www.anilearn.dev)
- [Twitter](https://www.twitter.com)
- and...

## How to add your own favicon?

In order to add your _Favicon_, You need to add a link tag inside the **head** tag, Like the following code:

```html
<link rel="icon" href="yourImage.png"></link>
```

Your code should look something like this:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <link rel="icon" href="yourImage.png"></link>
  </head>
  <body></body>
</html>
```

## That is how you can add your _Favicon_
