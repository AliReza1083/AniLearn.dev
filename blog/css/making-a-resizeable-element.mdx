# CSS Resize Property 

The resize CSS property determines whether an element can be resized, and if so, in which direction.


## 1. Horizontally

Setting the value of the **_resize_** property to horizontal will allow the element to scale horizontally only.

```css:style.css
	.resizable {
		resize: horizontal
	}
```

> Example [codepen.io](https://codepen.io/abbas-roholamin/pen/QWVgbjy)




## 2. Vertically

Setting the value of the **_resize_** property to vertical allows the element to be scaled vertically only.


```css:style.css
	.resizable {
		resize: vertical
	}
```

> Example [codepen.io](https://codepen.io/abbas-roholamin/pen/eYLRNvG?editors=1100)


## 3. Both

If you want the element to resize both horizontally and vertically, set the value of the **_resize_** property to both.

```css:style.css
	.resizable {
		resize: both
	}
```

> Example [codepen.io](https://codepen.io/abbas-roholamin/pen/gOdRpKp)


<Tips>
  resize does not apply in the following cases:

	1. inline element
	2. Block elements with overflow property set to visible
</Tips>
