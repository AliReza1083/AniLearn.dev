# CSS In-line Internal External Style sheet

There are 3 ways of writing **CSS styles**

- Inline CSS
- Internal CSS
- External CSS

## Inline CSS

It is clear by its name, You can add your styles inside a element tag, like the following code:

### Example 1

```html:index.html
<h1 style="font-size: 1em; color: red;">...</h1>
```

### Example 2

```html:index.html
<div class="width: 300px; height: 300px;">
  <p style="color: blue;">...</p>
</div>
```

<Warning>
  We can not use pseudo-elements and pseudo-classes in **inline CSS**
</Warning>

## Internal CSS

You can add you styles inside the head tag, like the following code:

```html:index.html
<html>
  <head>
    <style>
      h1 {
        color: red;
        font-size: 24px;
      }
      ...;
    </style>
  </head>
  <body>
    <h1>...</h1>
  </body>
</html>
```

<Success>
  We can use pseudo-elements and pseudo-classes in **inline CSS**
</Success>

## External CSS

You can make a file for CSS, and connect your **\*.html** file to your **\*.css** file.

### How to connect them?

You can use the following code inside the head tag.

```html:index.html
<link rel="stylesheet" href="*.css" />
```

```html:index.html
<html>
  <head>
    <link rel="stylesheet" href="*.css" />
  </head>
  <body>
    <h1>...</h1>
  </body>
</html>
```

```css:style.css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

h1 {
  color: blue;
}

p {
  font-size: 24px;
}
```

_**\*** represents your file name._
