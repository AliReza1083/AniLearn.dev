# Random Number

Have you ever asked yourself, how will I be able to get a random number? If so, then you will learn by the end of this blog.

## How to get a random number in JS?

`Math.random()` always returns a number lower than 1.

```js:app.js
const randNum = Math.random();
// 0.33508977219265534
// 0.6803118467078817
// 0.8607315010012833
// ...
```

To fix the problem, You must multiply it with a number.

```js:app.js
const randNum = Math.random() * 100;
// 52.622168316210114
// 42.53375359884042
// 19.23626329198578
// ...
```

If you see the example above, We are getting a number with its decimal.

To remove the decimal, we can use `Math.floor`.

`Math.floor()` & `Math.random()` are used to give us a random number without decimals.

```js:app.js
const randNum = Math.floor(Math.random() * 100);
// 28
// 65
// 56
// ...
```
