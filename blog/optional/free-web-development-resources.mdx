# Free Web Development Resources

Have you ever considered that you can learn web development for free, without spending any money?
This blog can help you to access the necessary resources.

## Roadmaps

- HTML `(Hyper Text Markup Language)`
- CSS `(Cascading Style Sheets)`
- JS `(JavaScript)`
- Visual Control
- Nodejs
- Reactjs

HTML is standard markup language for web pages.

#### Resources

- [freecodecamp](https://www.youtube.com/watch?v=pQN-pnXPaVg)

  > Learn the basics of HTML5 and web development in this awesome course for beginners.

  > Easy Learning with HTML "Try it Yourself" <br />
  > With our "Try it Yourself" editor, you can edit the HTML code and view the result

### <img src="https://cdn-icons-png.flaticon.com/512/732/732190.png" class="icon" /> CSS

CSS is the language we use for styling the HTML elements.

#### Resources

- [w3schools](https://www.w3schools.com/Css/)

  > This CSS tutorial contains hundreds of CSS examples. <br />
  > With our online editor, you can edit the CSS, and click on a button to view the result.

  > Learn CSS in this full course for beginners. CSS, or Cascading Style Sheet, is responsible for the styling and looks of a website.
